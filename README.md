# HaqDar — हक़दार
### Voice-first civic memory system for government scheme discovery

🔗 **Live Demo**: https://haqdar-app.netlify.app

---

## What it does
HaqDar helps undocumented women, domestic workers, and daily wage workers discover which government schemes they qualify for — in their own language, on any phone, with zero login required.

The user speaks or types their situation in Kannada, Hindi, or English. HaqDar's AI agents extract their profile, search a vector database of government schemes, and return personalized matches with a plain-language explanation and document checklist.

## The problem
490+ central and state government schemes exist in India. Most eligible women never access them due to:
- Language barriers
- Lack of awareness
- Documentation confusion
- No internet or smartphone

## Tech stack
| Layer | Technology |
|---|---|
| Frontend | HTML, CSS, JavaScript (single file) |
| AI responses | Llama 3.1 via Groq API |
| Vector memory | Qdrant Cloud |
| Voice input | Web Speech API |
| Deployment | Netlify |

## Features
- 🎤 Voice input in Kannada, Hindi, English
- 🧠 AI-powered scheme matching with Qdrant vector search
- 📋 Document checklist for each scheme
- 📲 WhatsApp share — send results to family
- 🔒 Zero login, zero data stored
- 📱 Mobile first design

## Schemes covered
- PM Jan Dhan Yojana
- Janani Suraksha Yojana
- Karnataka Gruha Lakshmi
- PM Ujjwala Yojana
- Ayushman Bharat PM-JAY
- PM Awas Yojana Gramin
